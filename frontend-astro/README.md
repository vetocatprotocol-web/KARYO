# KARYOGIT Frontend - Astro

Frontend untuk platform KARYOGIT, dibangun dengan **Astro** - static site generator yang optimal untuk GitHub Pages.

## рџљЂ Quick Start

### Prerequisites
- Node.js >= 18
- npm atau yarn

### Installation
```bash
cd frontend-astro
npm install
```

### Development
```bash
npm run dev
```

Akses di `http://localhost:3000`

### Build untuk Production
```bash
npm run build
```

Output akan tersimpan di folder `dist/` yang siap di-deploy ke GitHub Pages.

## рџ”— Integration

### API Integration
Frontend terhubung dengan API backend di `http://localhost:3000/api`. Konfigurasi ada di `src/lib/api.js`

Endpoints yang digunakan:
- `GET /api/projects` - Get projects list
- `GET /api/projects/:slug` - Get project detail
- `GET /api/developers` - Get developers

### Sample Data
Saat ini menggunakan sample data untuk development. Saat API tidak tersedia, akan fallback ke sample data.
