# Astro Starter Kit: Minimal

```sh
npm create astro@latest -- --template minimal
```

This is a bare-bones Astro project that has had all the example code removed.

## 🚀 Project Structure

Inside of your Astro project, you'll see the following folders and files:

```
/
├── public/
├── src/
│   └── pages/
│       └── index.astro
└── package.json
```

Astro looks for `.astro` or `.md` files in the `src/pages/` directory. Each page is exposed as a route based on its file name.

There's nothing special about `src/components/` — it's just where we like to put any Astro/React/Vue/Svelte/Preact components.

Any static assets, like images, can be placed in the `public/` directory.

## 🧑‍🚀 Getting started

1. Install dependencies: `npm install`
2. Start the dev server: `npm run dev`
3. Open browser to `http://localhost:3000`
4. Start editing! Your site will hot-reload as you make changes.

## 📖 How do I customize this setup?

### Styling

If you are using popular styling frameworks like Tailwind, Styled Components or CSS Modules, follow their respective integration guides.

If you are not using a particular styling framework, you can create `.css` files in your `src/` and import them in your components.

### Add Components

See `src/components/` for examples.

### Add integrations

Integrations add new functionality to your project. Head to `https://astro.build/integrations` to learn about Astro Integrations.
