# KARYOGIT Technology Stack

## Frontend
- **Framework**: Vanilla JavaScript + Astro (optional)
- **Hosting**: GitHub Pages
- **CDN**: Cloudflare
- **Styling**: CSS3 + SCSS (optional)

## Backend
- **Runtime**: Node.js 18+
- **Framework**: Express.js
- **Hosting**: Render.com (Free Tier)
- **Database**: Supabase PostgreSQL
- **Auth**: Supabase Auth + GitHub OAuth

## Storage & CDN
- **File Storage**: Wasabi S3 (atau Backblaze B2)
- **CDN**: Cloudflare
- **Cost**: ~$6/month untuk storage atau GRATIS dengan B2 (10GB gratis)

## Services
- **Database**: Supabase (PostgreSQL, Realtime, Auth)
- **Email**: Resend (3000 emails/month gratis)
- **Analytics**: Umami (self-hosted di Render)
- **Monitoring**: UptimeRobot (50 monitors gratis)

## CI/CD
- **Version Control**: GitHub
- **CI/CD**: GitHub Actions (2000 menit/bulan gratis)
- **Deployment**: Render (auto-deploy) + GitHub Pages

## Development Tools
- **Package Manager**: npm
- **Version Control**: Git + GitHub
- **Code Quality**: ESLint
- **Testing**: Jest (optional) + Playwright (optional)
